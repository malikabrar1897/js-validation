# JavaScript Client Side Validation

## Aim
To perform client-side validation using JavaScript for registration and login pages.

## Features
- Username validation
- Email validation
- Password validation
- Error message display
- Responsive design using Bootstrap

## Technologies Used
- HTML5
- Bootstrap 5
- JavaScript
